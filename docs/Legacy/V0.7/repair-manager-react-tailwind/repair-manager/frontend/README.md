# Repair Manager Frontend - React + Tailwind CSS

Tailwind CSS has been added only to the React frontend. The MERN backend/API structure is not changed.

## Run

```bash
cd frontend
npm install
npm start
```

## Tailwind files added

- `tailwind.config.js`
- `postcss.config.js`
- Tailwind directives in `src/styles.css`
- Tailwind dependencies in `package.json`

Existing CSS classes were kept so the current design remains the same. You can now use Tailwind classes inside any React component.
